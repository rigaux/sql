  2. Les noms des touropérateurs Japonais qui ont vendu des billets pour le stade de Marseille? (2 points)
    Solutions:

  3. Les noms des touropérateurs avec le nombre des billets vendus ? (2 points)
    Solutions:
    algebre pas possible 
    select Opérateurs.NomOp, count(distinct NumBill)
  from Opérateurs, Billets

  4. Les stades oů il y a plus de billets vendus que de places disponibles? (3 points)
    Solutions:
    algebre pas possible 
    select NomStade
  from Stades
 where Places < (select count(distinct NumBill)
                   from Billets
                  where Billets.NomStade = Stades.NomStade)
  5. Les noms des touropérateurs ayant vendu le plus de billets ? (2 points bonus)
    Solutions:
    algebre pas possible 
    select NomOp
  from Billets
 group by NomOp
having count(NumBill) >= all
                 (select count(NumBill)
                    from Billets
                   group by NomOp)
