2   Dépendances fonctionnelles (4 points)

Soit un schéma relationnel avec les dépendances fonctionnelles suivantes :

R(A, B, C) FR={A ® B}
S(C, D, E, F) FS={C ® D; C ® E; DE ® F; F ® C; F ® D; D ® F }

Question 2.1 :
(1 pt) Calculez les clés des relations R et S.

Solution :
  • la clé de R: (AC)
  • les trois clés de S: C, F et D

Question 2.2 :
(1 pt) Est-ce que S est en 3e forme normale?

Solution : OUI

Question 2.3 :
(1 pt) Soit un sous-ensemble de dépendances fonctionnelles G Ì FS :
C ® D
C ® E
F ® C
D ® F

Est-il possible d'obtenir toutes les dépendances fonctionnelles dans FS à partir de G en appliquant les axiomes d'Armstrong ? Justifiez votre réponse.

Solution : OUI. On obtient
  1. DE ® F par extension de D ® F en DE ® FE et décomposition.
  2. F ® D par transitivité (F ® C et C ® D)

Question 2.4 :
(1 pt) Est-ce que FS est une couverture minimale ? Justifiez votre réponse (en utilisant éventuellement le résultat de la question précédente).

Solution : NON, car G+ = FS+ et G Ì FS.
