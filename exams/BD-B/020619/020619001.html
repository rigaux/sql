1   Conception de schéma (3 points)

Soit les tables relationnelles suivantes (les attributs qui forment une clé sont soulignés) :
Produits
(code, marque, desig, descr)
  • code : identifiant du produit sous forme de chaîne de caractères (par exemple : 'PV1567-234')
  • marque : marque du produit (par exemple : 'Sony')
  • desig : désignation du produit (par exemple : 'platine laser')
  • descr : description détaillée du produit

PrixFour
(code, nom, prix)
  • code : code du produit
  • nom : nom du fournisseur du produit (par exemple : 'Darty')
  • prix : prix (en euros) du produit chez le fournisseur

NoteMag
(code, titre, note)
  • code : code du produit
  • titre : titre du magazine
  • note : note entre 1 et 10 du produit dans le magazine
  1. Dessinez un schéma entité-association qui correspond au schéma relationnel précédent.
