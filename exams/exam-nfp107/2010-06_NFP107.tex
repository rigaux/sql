\section{Concurrence (5 points)}

Soit les deux programmes de locations suivants :

\textbf{Programme P1 :}
\begin{itemize}
\item 
\scriptsize
\begin{verbatim}
Location (Num_Lot, date_debut, date_fin, Num_Locataire)
  var tmp
begin
  Start
  //Lecture des locations du lot entre date_debut et date_fin
  tmp = Read (Location, Num_Lot, date_debut, date_fin)
  if (tmp == null)
    //Lecture des travaux du lot entre date_debut et date_fin
    tmp = Read (Travaux, Num_Lot, date_debut, date_fin)
    if (tmp == null)
      //Ajout de la location
      Write (Num_Lot, Num_Locataire, date_debut, date_fin)
      Commit
    else
      Abort
  else
    Abort
end
\end{verbatim}
\end{itemize}

\textbf{Programme P2 :}
\begin{itemize}
\item 
\scriptsize
\begin{verbatim}
Travaux (Num_Lot, date_debut, date_fin)
  var tmp
begin
  Start
  //Lecture des locations du lot entre date_debut et date_fin
  tmp = Read (Location, Num_Lot, date_debut, date_fin)
  if (tmp == null)
    //Ajout des travaux (il peut y avoir plusieurs travaux sur un mźme lot)
    Write (Travaux, Num_Lot, date_debut, date_fin)
    Commit
  else
    Abort
end
\end{verbatim}
\end{itemize}

Soit l'exécution concurrente suivante:

\textbf{H:} $r_1[x] r_2[x] r_1[y] w_2[y] r_3[x] w_1[x] r_3[y] c_1 c_2 a_3$

\begin{enumerate}
\item (1 points) Identifier pour chaque transaction ą quelle programme (P1 ou P2) il peut correspondre.

  
\item (2 points) Identifier les conflits et donner le graphe de sérialisation. Cette histoire est-elle sérialisable ?
 
\item (2 points) Trouver l'exécution produite par verrouillage ą deux phases
simple si les verrous d'une transaction sont toujours relāchés aprčs sa
validation ou son annulation. Les opérations bloquées en attente de verrou
s'exécutent en priorité quand le verrou devient disponible, en respectant
l'ordre de blocage. En cas d'interblocage, la transaction
\linebreak \underline{la plus ancienne est annulée et non réinsérée} (la plus
vieille transaction est annulée au moment oł l'interblocage apparaīt).

 
\end{enumerate}
